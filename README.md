# Egg App Backend

`go run ./cmd/server`

## Environment variables

| Name               | R | Description                                                                      | Default      |
|--------------------|---|----------------------------------------------------------------------------------|--------------|
| **RUNTIME**        | F | `development`, `production`                                                      | `production` |
| **MONGODB_URI**    | T | e.g. `mongodb+srv://<user>:<password>@<cluster-url>?retryWrites=true&w=majority` |              |
| **TELEGRAM_TOKEN** | T |                                                                                  |              |
